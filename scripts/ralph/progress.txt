## Codebase Patterns
- dotnet test may fail with socket permission errors in sandboxed environments; use --no-restore and disable sandbox for MSBuild named pipes
- Test types for AsyncAPI tests inherit from Event (e.g., `public class MyEvent : Event { public MyEvent() : base(Guid.NewGuid()) { } }`)
- Always set `DisableAssemblyScanning = true` in test options unless testing assembly scanning specifically
- FakeItEasy is used for mocking interfaces (e.g., `A.Fake<IAmASchemaGenerator>()`)
- Schema refs use `#/definitions/` (NJsonSchema) or `#/$defs/` (modern JSON Schema) prefixes
- The RewriteEmbeddedSchemaRefs method rewrites both `#/definitions/` and `#/$defs/` patterns
