InitializeFromAttributeParams
Initializes from the RequestHandlerAttribute attribute parameters. Use when you need to provide parameter information from the attribute to the handler. Note that the attribute implementation might include types other than primitives that you intend to pass across, but the attribute itself can only use primitives. You couple the handler to a specific attribute using this as you need to know about the parameters passed, so this is really only appropriate for an attribute-handler pair used to provide orthogonal QoS to the pipeline.
