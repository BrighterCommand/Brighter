CommandProcessor..::..Post<(Of <(<'T>)>)> Method

Posts the specified request. The message is placed on a task queue and into a message store for reposting in the event of failure. You will need to configure a service that reads from the task queue to process the message Paramore.Brighter.ServiceActivator provides an endpoint for use in a windows service that reads from a queue and then Sends or Publishes the message to a CommandProcessor within that service. The decision to [Send{T}] or [Publish{T}] is based on the mapper. Your mapper can map to a Message with either a MT_COMMAND , which results in a [Send{T}(T)] or a MT_EVENT which results in a [Publish{T}(T)]
